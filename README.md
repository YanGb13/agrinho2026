# Agrinho 2026 - Uso Consciente da Água na Agricultura

## Tema
Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.

## Assunto Escolhido
**Uso Consciente da Água na Agricultura**

## Sobre o Projeto
Projeto desenvolvido para o Concurso Agrinho 2026 na categoria Programação Web.  
O site apresenta a importância da economia de água na agricultura e inclui uma calculadora interativa.

## Objetivo
Mostrar como pequenas mudanças nas técnicas de irrigação podem gerar grande impacto ambiental e econômico.

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
- Inteligência Artificial (apoio)

## Funcionalidades
- Design responsivo (funciona no celular)
- Calculadora de economia de água
- Informações educativas sobre sustentabilidade
